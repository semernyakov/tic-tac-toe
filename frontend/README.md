# Tic-tac-toe frontend

Этот каталог содержит фронтенд на React + TypeScript и настроен для запуска через Vite.

Локальные команды (в каталоге `frontend`):

Установка зависимостей:

```bash
npm install
```

Запуск dev-сервера:

```bash
npm run dev
```

Сборка для продакшена:

```bash
npm run build
```

Просмотр собранного билда:

```bash
npm run preview
```

Примечания:
- Используется `src/index.tsx` как точка входа.
- Если у вас нет Node.js/NPM, установите их: https://nodejs.org/
- В `App.tsx` используются переменные для Telegram бота — замените их на реальные значения или отключите отправку сообщений.
